# Tela de Login com PHP POO

Projeto desenvolvido para praticar Programação Orientada a Objetos em PHP. A interface muda dinamicamente de acordo com o tipo de usuário (aluno ou professor).

## Funcionalidades

- Login com validação de credenciais
- Identificação do tipo de usuário (aluno ou professor)
- Interface personalizada com cor de fundo diferente para cada tipo
- Controle básico de sessão

## Tecnologias Utilizadas

- PHP (POO)
- HTML e CSS
- XAMPP (Ambiente local)
